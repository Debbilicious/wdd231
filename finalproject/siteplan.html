Debby's Fashion Collection - Site Plan

1. Site Name

Debby's Fashion Collection (DFC)

The name "Debby's Fashion Collection" represents my personal fashion boutique offering premium fashion items. The acronym "DFC" creates a memorable brand identity that's modern and sophisticated. It reflects exclusivity and quality, appealing to fashion-conscious customers seeking stylish pieces.

Optional domain: dfc-fashion.com or debbyfashion.com

Why I Chose This Subject

I chose a fashion-themed website because I have personal experience running a fashion boutique. I used to sell clothing, shoes, accessories, and jewelry, managing the business operations and customer relationships. Creating this site allows me to combine my real-world fashion business experience with web development, while demonstrating all the required JavaScript and technical skills for the project.

2. Site Purpose

DFC Fashion is a premium fashion catalog website showcasing my curated selection of 20+ clothing, shoes, accessories, and jewelry items. The site provides:

  • Home Page (index.html): Introduction to the collection with hero section and 6-8 featured trending items
  • Gallery/Collection Page (gallery.html): Displays all 20+ fashion items dynamically loaded from a JSON data source with category filtering and search functionality
  • About Us Page (about.html): My personal story in the fashion business, brand mission, and passion for curating quality fashion pieces
  • Contact Page (contact.html): Contact form for customer inquiries and business information
  • Thank You Page (thankyou.html): Form submission confirmation page (does not count toward 3-page requirement)

Dynamic Features

The site demonstrates JavaScript proficiency through:

  • Data fetching from JSON file using Fetch API with try/catch error handling
  • Dynamic content generation displaying 20+ items with 4+ properties each (image, name, price, category, description)
  • Local storage for wishlist/favorites functionality
  • Modal dialogs for detailed product information
  • DOM manipulation and event handling (clicks, filters, sorting)
  • Array methods (filter, map, forEach) for data processing
  • Template literals for dynamic HTML generation
  • ES Modules for code organization

3. Target Audience Scenarios

Scenario 1:

"I want to see the latest fashion items with their prices before making a purchase decision."

Answer: Browse our complete collection on the Gallery page where all 20+ items are displayed with high-quality images, detailed descriptions, pricing, and category information. Use filters to narrow by category (clothing, shoes, accessories, jewelry).

Scenario 2:

"I want to save my favorite items so I can review them later during my shopping session."

Answer: Click the heart icon on any product to add it to your wishlist. Your selections are saved using local storage and remain available throughout your browsing session.

Scenario 3:

"I want to learn about the boutique owner and contact them with questions about specific items."

Answer: Visit the About Us page to learn about my fashion business journey and experience. Then use the contact form on the Contact page to submit inquiries, or find direct contact information including email and phone number.
